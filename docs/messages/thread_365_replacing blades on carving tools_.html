From: edwebbe (edwebbe@yahoo.com) | Date: 2004-09-30 17:12:22 UTC
Hi Jodie-
I just use a straight exacto blade, but I replace it every
third stamp or so and I can tell the difference. They are a lot
cheaper than carving tool blade sets. For a carving tool I wouldn't
change it as often, but I would change it if I noticed tearing. Can
you just buy one nib? The stores around me only sell sets.
