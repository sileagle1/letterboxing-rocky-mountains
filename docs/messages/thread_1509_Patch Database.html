Patch Database

From: Autumn (autumnkat@gmail.com) | Date: 2008-04-25 21:05:29 UTC
To help those that are ordering patches keep track of who wants how
many - I've added a database to our website. If you want patches -
simply add a record to the database with your information. Clicky:

http://groups.yahoo.com/group/RMLetterboxing/database?
method=reportRows&tbl=1

If the link doesn't work for you - here's the step by step.

* Go to the RMLB webpage at
http://groups.yahoo.com/group/RMLetterboxing
* On the left side of the screen, click Database
* Once the database screen opens, click the Patch Database
* Click Add Record
* Fill in your details
* Eat Pie.

Since some folks have already posted their interest, I've put some of
you guys in - if you're already on the list - just update your
information so Aljan can be in touch.

Lemme know if ya'll get stuck.
Autumn
