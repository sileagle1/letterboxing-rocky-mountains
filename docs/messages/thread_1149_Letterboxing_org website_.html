This message was posted by Choi on the big talk list for LbNA:

"The Letterboxing North America site will be offline until we can
move to a new hosting service. Unfortunately, the low cost web
hosting that we've been using can't accept the increasing load that
we're putting on the shared server as letterboxing has grown over the
years. More details on where we'll go next soon."

**

Since it looks like it might be awhile, here's some info sources in
the interim:

Link to LbNA Yahoo talk list for anyone who wants updates and news on
this issue. (I just also added this to the links section on the RM
talk list)

http://groups.yahoo.com/group/letterbox-usa/

Other than Atlasquest, I'm not aware of any other comprehensive clue
lists. However, you can view cached clues on Atlasquest for boxes
which have links to LbNA (in addition to clues actually posted on
AQ). See this discussion for more info:

http://www.atlasquest.com/boards/message.html?
gMsgId=31093&gThreadId=31093

Hope this helps. Too bad about the timing...right before the weekend.
