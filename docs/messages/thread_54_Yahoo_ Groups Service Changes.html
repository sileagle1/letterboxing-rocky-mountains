Yahoo! Groups Service Changes

From: Autumn (Autumn@shadowslight.com) | Date: 2003-08-14 22:27:27 UTC-06:00
Just a notice that beginning August 21, 2003, message attachements will not
be able to be viewed on the message board. They will still arrive if you
receive "individual emails" but if you get a digest, or use the web to view
the messages, you will NOT see attachments.

The best solution to this is to simply NOT use attachments! :) They're not
safe anyway on lists such as these. So, if you've got something interesting
to share, either cut and paste, or save the interesting tidbit in the
"files" section of the list.

Here's the catch: To access access a group's Files and Photos area, you
need to have a "web membership" effective that same date. To make sure you
have web memberships to all of your groups, please visit this page:
http://groups.yahoo.com/memwiz
Just run the wizard, it's easy!

These changes (and a few others) are affecting ALL yahoogroups, not just
ours. So - you may see messages like this if you subscribe to more than one
list.

Cheers!
