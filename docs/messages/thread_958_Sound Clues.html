Es is referring to the Big Brain Box, whose clues are a mini-CD that
floats around from box to box like cuckoo clues. However, you don't
need any special software for that, they should play in a normal CD
player.

I think most people who have a computer these days have some kind of
media player, although which one is a different story. If you're
going to go the media player route, I suggest you make sure the
sound clip is available in a few of the major formats.
