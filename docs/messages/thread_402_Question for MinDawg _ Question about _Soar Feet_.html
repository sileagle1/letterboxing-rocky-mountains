I expect a ranger would just pull the box, not write a nasty note and
take the stamp. Probably some muggle found it by accident.
Also, the only time you have to get a permit from Jefferson County is
if the box is in one of their Open Space Parks. Otherwise they don't
have any authority. If it's in a CO State Park, the current policy is
to verbally clear it with the local ranger when placing it - no need
to go through any state bureaucracy.
