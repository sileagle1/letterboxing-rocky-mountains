Welcome to the Rocky Mountain Lbox List!

From: Autumn (Autumn@shadowslight.com) | Date: 2003-07-10 08:37:28 UTC
Hey there!
I don't know about the rest of you - but sometimes reading the
hundreds of emails regarding New England letterboxes can be a bit
intimidating.

So - I've created this list so that we can discuss letterboxes a bit
closer to home, post news on missing boxes, or beg for help on
clues.

At some point, I'd like to have a gathering of local/regional
letterboxers for an exchange, and perhaps some stamp carving and
boxing! We have some great open spaces, parks, and forest areas that
are just screaming for more boxes.

Thanks for signing on -
Cheers,

Autumn
