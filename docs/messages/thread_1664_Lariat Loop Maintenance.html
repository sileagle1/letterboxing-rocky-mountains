Lariat Loop Maintenance

From: front range hiker (frhiker@comcast.net) | Date: 2009-02-15 22:30:37 UTC
I am spending this month doing maintenance on my contributions to the
Lariat Loop series. I'm adding a second stamp to each box, now that my
carving skills are a little better than they were in winter 2007.
Hopefully these second stamps will inspire new carvers to keep
practicing ;-)

The Yellow box was missing, so I have replanted it in a nearby tree
stump. Please make sure you reprint clues before attempting the yellow
box. I recarved the original image and added a second one to this box.

To see which boxes have new stamps, look for a "found date" of February
2009 on Atlas Quest.

Happy hunting!
