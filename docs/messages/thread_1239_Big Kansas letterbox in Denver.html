From: HER (fauxsum@hotmail.com) | Date: 2007-01-18 20:47:30 UTC
I did that box right after this out of town visitor placed it. When
I sent a status update I mentioned that I was concerned about its
longevity and his response was basically "we'll see."

In the summer I bike home right by this box and would stop and peek
to see if it was still there every once in a while. Then the big
giant mondo flooding happened ...in erm... late summer, I think it
was. I know I lost my own Denver Skatepark box to flooding (since
replaced).

So....while I can't say for sure...even if you do manage to get that
audio file to work (can't help with that - and I'm wondering if it
has been pulled because he knows the box is probably gone??)...to be
honest I'd be very surprised if that box is still there.

With the current weather we've had "forever" in Denver, it'd be
awhile until I could check on it - but next time I'm that way I'll
check it out and post here when I know more. I'm thinking that'll
probably be July 4th at the rate we're going....

Not sure if that helps but that's what I know.
