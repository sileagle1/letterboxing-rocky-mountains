From: edwebbe (edwebbe@yahoo.com) | Date: 2005-11-03 23:20:25 UTC
Well, it's not all that exciting. I just thought I should let everyone
know that clues and the hiding spot have changed for the Three Keys
letterbox in Durango, Colorado.

So if anyone was planning on looking for that, they need to print up a
new sheet.

There were just too many wrong turns possible when solving the first
set of clues. And the hiding spot made it a little hard to be discreet
when re-hiding.

I would like to congratulate those people who solved the first set and
welcome them to stamp in again if they get a chance to visit it's new
location.

Happy Trails,

y-nought
