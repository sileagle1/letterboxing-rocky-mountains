Re: envelopes for patches

From: front range hiker (frhiker@comcast.net) | Date: 2008-06-10 01:46:31 UTC
I checked the USPS.GOV web site. Postage for a large envelope is 83
cents (weighing up to 1 ounce) and $1 for up to 2 ounces. If the
envelope is square, add 20 cents for "non-machine processing". I
don't know how much a patch weighs, but I think you could probably
mail 2-3 of them for the 1 ounce price.

Hope this helps.

FRH
