Now that Judge-nought has spilled the proverbial beans you all might

original "Dreadnoughts", a branch of the Knights Templar that stole

from the Dread side of the family, but alas, "once a pirate always a
pirate". And after meeting Dixie everyone wants to be a Dread anyway.

Code is really the same as the Pirate's Code (which is more of a
guideline, really).

If this makes sense to anybody, please seek professional help.
